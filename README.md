# ow-ez-chat
Overwatch用多言語チャット発行ツール。英語・韓国語・中国語の定型文をワンクリックでコピーし、素早いチーム連携をサポートします。

# OW多言語チャット発行ツール (Overwatch Multilingual Chat Generator)

![Version](https://img.shields.io/badge/version-1.56-orange)
![License](https://img.shields.io/badge/license-MIT-blue)

**「あーあ、韓国語でコミュニケーション取れたら勝てるのに」**
そんなオーバーウォッチプレイヤーのために開発された、ワンクリックで多言語チャットをコピーできる支援ツールです。
追加してほしい項目、不具合等ありましたら、ボ林(https://x.com/boba_yashi)にDMください。

## 🚀 特徴

- **多言語対応**: 日本語の定型文を、英語・韓国語・中国語に一瞬で変換。
- **ワンクリックコピー**: クリックするだけでクリップボードにコピー。あとはゲーム内で貼り付けるだけ。
- **完全ローカル保存**: 並び替えた位置や編集したタイトルは、お使いのブラウザに自動保存されます。

## 🛠 使い方

1. **フレーズを選ぶ**: 使いたいメッセージをクリックします。
2. **コピー完了**: 自動的にクリップボードにコピーされ、画面に「COPIED!」と表示されます。
3. **ゲーム内で送信**: Overwatchのチャット欄（Enterキー）を開き、`Ctrl + V` で貼り付けて送信してください。
